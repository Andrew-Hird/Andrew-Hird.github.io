Javascript Language

August 4th 2016
parentheses ()

Parentheses in javascript are where you hold the parameter names of function and, in order to be able use them when you call the function later. You use the parentheses to define a function add(x, y) and also to call a function add(2, 3).

They also surround conditional statements in other types of functions such as if, else if and for loops, for example for (var i = 0; i < 10; i++).

You can also use them as you would in maths to determine the order of operations, for example (5 + 4) / 2.

brackets []

Square brackets in javascript are used to hold the value of an array (a kind of list) and also hold the index number if you want to access the array (the position within the array). For example colours [red, blue, green] and colours[2] = green .

braces {}

Curly braces are used to define the start and end of a function and separate the code into blocks within a function, which not only makes it easier for us to read but also tells javascript the structure of the code.

They are also used to hold the key and value when defining an object. fruit {apple: red, banana: yellow, pear: green}

'single quotes' & "double quotes"

Both have the same use and it is up to the coder what their preference is. If something is within the quotation marks it tells the browser to treat what you have written as plain text and not code. Though both can be used, you need to use the same mark to close with as you opened with.
